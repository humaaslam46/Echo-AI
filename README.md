# Echo-AI
Name :  Echo - AI StoryTelling Companion

📖 Echo — AI Storyteller’s Companion

Elevo Pathways | Front-End Web Developer Internship | Task 5 (Bonus Task)



📌 Project Overview

Echo is a responsive marketing landing page designed for a fictional SaaS product:

> Echo — The AI-Powered Storyteller’s Companion
The goal of this project (Task 5) was to design and implement a modern, responsive landing page showcasing a tech product, while practicing layout design, responsive structure, and user experience principles.

Echo is a creative partner for writers, authors, and content creators that helps with:

Plot development

Character backstories

World-building

Idea generation



✨ Features

✅ Responsive layout (mobile, tablet, desktop)
✅ Hero section with engaging background image + CTA button
✅ Feature cards with icons & gradients
✅ Testimonials with user avatars
✅ Pricing section (Free + Pro) with styled cards
✅ Theme-aligned color scheme (Aqua, Gold, Hot Pink, Honeysuckle)
✅ Footer with social icons (Facebook, Instagram, YouTube)
✅ Smooth navigation bar (with hamburger menu on mobile)



🛠 Tech Stack

HTML5 → Page structure

CSS3 (Flexbox/Grid + custom styles) → Styling & responsiveness

TailwindCSS (utility classes) → Modern styling framework

JavaScript (Vanilla) → Mobile nav toggle

FontAwesome → Social & navigation icons

Unsplash → Hero and feature images
